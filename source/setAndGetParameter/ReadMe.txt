========================================================================
    CONSOLE APPLICATION : setAndGetParameter Project Overview
========================================================================

This is a simple example of how to find and connect an available Mech-Eye Device
and then get and set parameter data with the Mech-Eye Device.

You will learn how to:

* use Mech-Eye API,
* get and set parameter data.

How to build:

1. Copy setAndGetParameter folder to a location with Read and
   Write permissions (using the name <source>)
2. Open CMake
   2.1. Set Source code to <source>
   2.2. Set Binaries to <source>/_build or any other writable location
   2.3. Click Configure and Generate
3. Build project
4. Run setAndGetParameter
   4.1. Connect to a device
5. Run setAndGetParameter application
6. Add your code to run your code from this sample

The application will print out basic info about Mech-Eye Device.
